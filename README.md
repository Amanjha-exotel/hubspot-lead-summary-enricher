# PeopleStrong — Lead Intelligence & SDR Briefing Agent

Prototype mockup for the AI-powered inbound lead enrichment, intent scoring, and SDR routing agent.

## Stack
- React 18
- Vite 5

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect this repo in [vercel.com](https://vercel.com) for auto-deploy on push.

## Tabs
- **Lead Inbox** — Live lead feed with AI briefing panel. Click leads to switch.
- **Routing & Capacity** — SDR team load balancing + routing logic rules.
- **Agent Flow** — End-to-end 5-step agent process (Ingest → Enrich → Score → Brief → Route).
- **Performance** — Before/after metrics vs. pre-agent baseline.
